# 【清松手寫體】

「清松手寫體」是清松在稿紙上用原子筆手寫，透過「守寫字」網站產生出初稿字型檔，再透過字型軟體後製加工而成的手寫字型。

這裡分享的，並不是官方的版本哦，是Max修改過的，原版的清松手寫體，請在這一個網址下載：
https://github.com/jasonhandwriting/JasonHandwriting
